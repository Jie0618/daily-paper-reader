## 摘要
我们介绍了 SILAS，这是一个用于发现具有可证明有界轨迹的多项式常微分方程（ODE）的数据驱动框架。有界性通过由多项式 Lyapunov 函数定义的紧致吸收集来证明。我们利用多项式优化工具构建了一个适定的非凸优化问题，从而联合识别 ODE 向量场和 Lyapunov 函数。我们使用带有凸子问题的交替块坐标优化方案来解决该问题，其可行性由一种新型的模型无关初始化方法保证，该方法能从数据中识别出候选 Lyapunov 函数。我们的方法将先前针对具有吸收椭球体的二次 ODE 的方法扩展到了更广泛的 ODE 类别和吸收集。包含 100 多个示例的测试集表明，SILAS 能够为广泛的非线性动力系统恢复准确且可证明有界的 ODE 模型。

## Abstract
We introduce SILAS, a data-driven framework for discovering polynomial ordinary differential equations (ODEs) with provably bounded trajectories. Boundedness is certified by compact absorbing sets defined via polynomial Lyapunov functions. We jointly identify the ODE vector field and the Lyapunov function using a well-posed nonconvex optimization problem built using polynomial optimization tools. We solve this problem using an alternating block-coordinate optimization scheme with convex subproblems, whose feasibility is ensured by a novel model-agnostic initialization that identifies a candidate Lyapunov function from data. Our methods extend prior approaches for quadratic ODEs with absorbing ellipsoids to a significantly broader class of ODEs and absorbing sets. A suite of over 100 examples demonstrates that SILAS can recover accurate and provably bounded ODE models for a broad range of nonlinear dynamical systems.

*   **核心问题**：如何从观测数据中发现一个多项式 ODE 模型，使其不仅能拟合数据，还能在数学上**证明**其所有轨迹在时间趋于无穷时都保持在某个有界区域（吸收集）内。
*   **背景**：该研究结合了动力系统理论（Lyapunov 函数）、凸优化（平方和/SOS 编程）和算子理论（Koopman 生成元）。

### 2. 方法论
*   **核心思想**：利用 Lyapunov 函数 $v(x)$ 来定义一个紧致吸收集 $A = \{x : v(x) \le b\}$。如果能找到一个 $v(x)$ 满足特定的耗散不等式，则该 ODE 的解必有界。
*   **关键技术细节**：
    *   **联合优化**：将 ODE 向量场 $f(x)$ 的系数和 Lyapunov 函数 $v(x)$ 的系数作为变量，构建一个非凸优化问题。
    *   **SOS 约束**：利用多项式平方和（Sum-of-Squares）技术，将“函数非负”这一硬约束转化为半正定规划（SDP）问题，从而确保 Lyapunov 条件在全空间成立。
    *   **交替块坐标优化**：由于 $f$ 和 $v$ 在约束中是双线性耦合的（非凸），SILAS 采用交替优化策略：固定 $v$ 优化 $f$，再固定 $f$ 优化 $v$。
    *   **模型无关初始化**：这是 SILAS 的关键创新。它不依赖于预先识别的 ODE，而是利用数据通过类似扩展动态模式分解（EDMD）的方法，直接从数据中估计 Koopman 生成元的动作，从而初始化一个候选的 Lyapunov 函数，确保后续优化步骤的可行性。

*   **有界性保证**：SILAS 能够可靠地生成具有可证明有界解的模型，即使对于那些原始系统可能存在发散轨迹的案例，SILAS 也能捕捉其局部吸引子并保证有界。

### 8. 不足与局限
*   **维度灾难**：SOS 编程的计算复杂度随状态维度 $n$ 和多项式阶数 $d$ 呈组合爆炸式增长，目前难以直接处理数十维以上的高维系统。

*   **局部性**：虽然证明了全局有界，但吸收集的大小可能比实际吸引子大得多，且在数据未覆盖区域的拟合可能缺乏物理真实性。
