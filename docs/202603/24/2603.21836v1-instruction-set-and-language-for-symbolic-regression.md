## 摘要
符号回归（SR）中一个基本但很大程度上尚未解决的障碍是结构冗余：每个表达式有向无环图（DAG）都允许许多不同的节点编号方案，这些方案编码的是同一个表达式，每个方案都占据搜索空间中的一个独立点，并在不增加多样性的情况下消耗适应度评估。我们提出了 IsalSR（符号回归指令集与语言），这是一个表示框架，它将表达式 DAG 编码为紧凑两层字母表上的字符串，并计算一个剪枝规范字符串——一个完整的带标签 DAG 同构不变量——从而将所有等效表示折叠为单一的规范形式。

## Abstract
A fundamental but largely unaddressed obstacle in Symbolic regression (SR) is structural redundancy: every expression DAG with admits many distinct node-numbering schemes that all encode the same expression, each occupying a separate point in the search space and consuming fitness evaluations without adding diversity. We present IsalSR (Instruction Set and Language for Symbolic Regression), a representation framework that encodes expression DAGs as strings over a compact two-tier alphabet and computes a pruned canonical string -- a complete labeled-DAG isomorphism invariant -- that collapses all the equivalent representations into a single canonical form.

符号回归（SR）旨在从数据中发现闭式数学表达式。目前 SR 面临的一个核心障碍是**结构冗余（Structural Redundancy）**：
*   **组合爆炸**：一个具有 $k$ 个内部节点的表达式有向无环图（DAG），存在 $\Theta(k!)$ 种不同的节点编号方案，它们在数学上完全等价，但在搜索空间中却被视为不同的点。
*   **资源浪费**：现有的遗传编程（GP）或深度学习 SR 方法会重复评估这些等效结构，消耗大量计算资源却不增加种群多样性。

### 2. 方法论：核心思想与关键技术
IsalSR 的核心是将表达式 DAG 编码为指令序列，并定义了一套确保唯一性的规范化流程：
*   **双层指令集 ($\Sigma_{SR}$)**：包含指针移动（N/P/n/p）、边创建（C/c）和带标签节点插入（$V_\ell/v_\ell$）。这种设计确保了字母表上的**任何字符串都能解码为一个有效的 DAG**，无需复杂的语法检查。
*   **交换性分解**：为了简化同构定义，将非交换的二元运算（减法、除法）分解为一元运算（负号、倒数）与交换性二元运算（加法、乘法）的组合。仅保留 $POW$（幂运算）作为唯一的非交换二元算子。
*   **S2D 与 D2S 算法**：
    *   **S2D (String-to-DAG)**：确定性状态机，将指令流解码为 DAG。
    *   **D2S (DAG-to-String)**：贪心编码器，将 DAG 转换回字符串。
*   **剪枝规范字符串 (Pruned Canonical String)**：这是本文的核心算法。通过回溯搜索所有可能的插入顺序，寻找长度最短且字典序最小的字符串。为了提高效率，引入了基于 **6 分量结构元组 ($\tau$)** 的剪枝策略（利用节点 1-3 步内的入度/出度信息作为局部结构特征）。
