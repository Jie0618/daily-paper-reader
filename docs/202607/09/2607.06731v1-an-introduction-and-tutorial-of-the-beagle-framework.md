## 摘要
Beagle 框架是一个基于 GPU 的遗传编程框架，它通过利用 NVIDIA GPU，实现了在大种群规模下的高效遗传编程搜索。本技术指南介绍了 Beagle 框架，并提供了使用该框架解决符号回归问题的详细说明。

## Abstract
The Beagle framework is a GPU-based genetic programming framework that enables highly efficient genetic programming search using large population sizes by leveraging NVIDIA GPUs. This technical guide provides an introduction to the Beagle framework and provides detailed instructions for using the framework for symbolic regression problems.

*   **核心问题**：如何设计一个能够充分利用 NVIDIA GPU 并行计算能力的框架，以实现大规模种群的高效演化，并克服 GPU 编程中的内存分配和线程分歧等技术挑战。

### 2. 方法论
*   **核心思想**：采用 **CPU/GPU 协同演化架构**。CPU 负责遗传算子（变异、初始化、选择逻辑），而 GPU 负责最耗时的模型评估（适应度计算）。
*   **关键技术细节**：
    *   **GCL 语言**：使用自定义的“基因组计算机语言”（Genome Computer Language），这是一种基于**逆波兰表示法（RPN）**的**线性遗传编程（LGP）**语言。这种设计避开了树状结构在 GPU 上难以静态分配内存的问题。
    *   **任务分配与线程优化**：为了减少线程分歧（Thread Divergence），Beagle 让每个 CUDA 块（Block）对应一个模型个体，块内的每个线程处理该模型的一个适应度案例（Fitness Case）。
    *   **内存管理**：在 CPU 端引入“死池（Dead Pool）”机制，重用已死亡个体的内存空间，几乎消除了 C# 垃圾回收（GC）带来的性能抖动。
    *   **蒙特卡洛排名（Monte-Carlo Ranking）**：针对数百万规模的种群，不进行全局排序，而是通过随机采样 100 个个体来估计百分位数，从而实现高效的比例选择。
    *   **适应度函数**：默认采用**相关性适应度（Correlation Fitness）**，通过计算模型输出与目标值的线性相关性（$r^2$）来评分，对常数偏移具有鲁棒性。
