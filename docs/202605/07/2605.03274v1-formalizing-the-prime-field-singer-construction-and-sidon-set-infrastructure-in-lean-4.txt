Abstract 

Erdős Problem 30 asks for sharp asymptotics of the Sidon extremal function h(N ), and Singer’s construction is the classical source of lower-bound examples matching the main term. We present a Lean 4 formalization of Singer’s Sidon set construction for prime fields, together with reusable Sidon-set infrastructure for additive combinatorics. For every prime p, we prove the existence of a Sidon set modulo p2 + p + 1 

of cardinality p + 1 . The proof proceeds through a non-trivial algebraic chain: construction of the Galois field GF (p3), analysis of the trace kernel as a 2-dimensional subspace, a geometric argument via subspace intersections establishing the multiplicative Sidon property in the quo-tient group, and a combinatorial bridge transferring this to modular integer arithmetic. Around this central result, we develop a reusable Sidon set library for additive combinatorics. It comprises interval Sidon sets, modular Sidon sets, the extremal function h(N ), Lindström’s cross-difference inequality, a Johnson-route shift-incidence upper bound of the form h(N ) ≤ √N + N 1/4 + O(1) , exact representation-function identities, and unconditional two-sided h(N ) = Θ( √N ) bounds with exact floor-rounded finite statements for N ≥ 5. We further formalize a conditional reduction: subpolynomial prime gaps together with a full subpolynomial upper-error hypothesis for h(N ) imply the Erdős

# 1 Introduction 

1.1 Erdős Problem 30 

Let h(N ) = max {| A| : A ⊆ { 1, . . . , N } is Sidon }, where a set A of integers is Sidon (or a B2 set) if all pairwise sums a + b with a ≤ b (including a = b)and a, b ∈ A are distinct. Lindström [15] established the upper bound 

h(N ) ≤ √N + N 1/4 + 1 ,

sharpening the earlier Erdős–Turán [ 10 ] bound. Later work [ 2, 8] improves the constant in the N 1/4 term but not the exponent. On the lower side, Singer’s construction and prime-finding results give h(N ) ≥ √N − O(N θgap ).Thus an upper error O(N θup ) and this Singer–prime lower error combine to give h(N ) = √N + O(N max( θup ,θ gap )). Here θgap denotes the prime-gap error exponent and θup the upper-bound error exponent for h(N ). Erdős Problem 30 [ 9, 12 ] asks whether this can be improved to h(N ) = √N +Oε(N ε)

for every ε > 0. This remains open.

1.4 Scope and limitations 

We separate deliberate modeling choices from practical limitations. 

Modeling choices (deliberate scope decisions that do not weaken stated results): 

• Singer’s original theorem [ 19 ] applies to all prime powers q. We treat only q = p (prime), using Lean’s GaloisField p 3 type. This suffices for the asymptotic lower bound, since primes alone are dense enough for the transfer argument. 

• The conditional Erdős 30 reduction relies on a subpolynomial prime gap hypothesis (implied by Cramér’s conjecture [ 7] but weaker) which remains open. This is inherent: no unconditional proof of Erdős 30 is known. 

Practical limitations (results not yet provided): 

• Extension to q = pk would require constructing GF (p3k) (via 

GaloisField p (3*k) ) and establishing an explicit degree-3 field-3tower interface over the subfield GF (pk); that tower interface is outside the prime-field formalization reported here.

52 Mathematical Background 

2.1 Sidon sets 

A set A ⊆ Z is Sidon if whenever a + b = c + d with a, b, c, d ∈ A and 

a ≤ b, c ≤ d, then {a, b } = {c, d }. Equivalently, one may omit the ordering convention and allow the two possible ordered identifications; the Lean predicate uses this unordered collision form. For a positive integer M , we say A ⊆ Z is Sidon modulo M if M | (a + b) − (c + d) with a, b, c, d ∈ A

implies {a, b } = {c, d } (as integer pairs, not just residues). Modular Sidon immediately implies ordinary Sidon: if a + b = c + d as integers, then 

M | (a + b) − (c + d) = 0 , so the modular condition gives {a, b } = {c, d }. The converse requires a bounded embedding: 

Lemma 2.1 (No-wraparound, IsSidon.isSidonMod_of_interval ). If A

is Sidon with A ⊆ { 1, . . . , N } and M ≥ 2N − 1, then A is Sidon modulo M .

The bound ensures all pairwise-sum differences lie in {− (2 N − 2) , . . . , 2N −

2} ⊂ (−M, M ), so M -divisibility forces the difference to be zero, reducing to an ordinary collision. 

2.2 Singer’s construction 

A (v, k, λ )-difference set is a subset D of Z/v Z with |D| = k such that every nonzero residue is representable as exactly λ ordered differences di − dj ; a 

(v, k, 1) -set is called perfect . The λ = 1 condition implies the Sidon property in the cyclic group (at most one representation per difference); for Singer parameters, where k(k − 1) = v − 1, the converse also holds. Singer [ 19 ] proved that for every prime power q, the quotient 

GF (q3)×/GF (q)× is cyclic of order q2 + q + 1 , and the image of the nonzero trace-kernel elements ker (Tr GF( q3)/GF( q)) \ { 0} under the quotient map forms a (q2 + q + 1 , q + 1 , 1) perfect difference set of size q + 1 in 

Z/(q2 + q + 1) Z. Indeed, the trace kernel has q2 elements, so its q2 − 1

nonzero elements fall into (q2 − 1) /(q − 1) = q + 1 scalar orbits. (We state the background for general q; our formalization handles q = p prime; see §3for scope discussion.) The Sidon property rests on a geometric fact. Let V = ker ( Tr GF( q3)/GF( q)),a 2-dimensional GF (q)-subspace (by rank-nullity applied to the surjective trace). For an element α of the extension not lying in the embedded base field: 1. The scaled subspace αV differs from V : if αV = V , then V would be a proper α-invariant submodule of GF (q3), contradicting the no-invariant-subspace theorem. 2. Two distinct 2-dimensional subspaces of a 3-dimensional space intersect in dimension 1 (Grassmann formula). 63. Therefore V ∩ α−1V is 1-dimensional for such α ∈ GF( q3) \ GF( q).This forces any “product collision” in the quotient (i.e., [u] · [v] = [ w] · [x]) to degenerate into a pair equality, establishing the multiplicative Sidon property that is then transferred to additive modular arithmetic via a cyclic quotient group isomorphism (the abstract discrete-logarithm map). 

2.3 Known bounds on h(N )

The pair-difference counting argument gives h(N ) ≤ √2N + 1 . Lindström’s cross-difference inequality [ 15 ] sharpens this to h(N ) ≤ √N + N 1/4 + 1 .Balogh–Füredi–Roy [ 2] and Carter–Hunter–O’Bryant [ 8 ] subsequently im-proved the constant in the N 1/4 term, the latter giving h(N ) ≤ N 1/2 +0.98183 N 1/4 + O(1) ; these refinements do not change the exponent-level in-terface formalized here. For a broader annotated guide to the Sidon-sequence literature, see O’Bryant [ 18 ]. For the lower bound, Singer sets transferred to intervals via prime-finding results give h(N ) ≥ √N −O(N θ) where θ depends on the available prime gap theorem (Bertrand gives θ = 1 /2; Baker–Harman– Pintz [ 3 ] gives θ = 0 .2625 ). Bose–Chowla [ 6] is another classical algebraic source of finite Sidon sets; in the present development it appears only at the parameter-wrapper and transfer-interface level, while the construction theorem used for the formal lower-bound pipeline is Singer’s.

3.1 Algebraic core

for any v̸ = 0 , using the fact that α has degree-3 minimal polynomial over GF (p) (since [ GF (p3) : GF (p)] = 3 is prime, there is no intermediate field, so every α / ∈ GF (p) generates the full extension). If c0v + c1αv + c2α2v = 0

Theorem 3.3 (Intersection dimension) . Any two distinct 2-dimensional 

GF( p)-subspaces of GF( p3) intersect in dimension 1. 

This follows from the Grassmann formula dim (V + W ) + dim (V ∩ W ) = dim V + dim W . Since V̸ = W are both 2-dimensional, dim (V + W ) = 3 ,hence dim( V ∩ W ) = 2 + 2 − 3 = 1 .

3.2 Quotient-group Sidon property 

We define the canonical group homomorphism singerMk : GF (p3)× →

GF (p3)×/GF (p)× and write [u] for the coset u · GF (p)×. Multiplication below denotes field multiplication in GF( p3). We prove: 

Theorem 3.4 (Singer quotient Sidon) . Let u, v, w, x ∈ ker ( Tr ) \ { 0} and 

α ∈ GF (p)× satisfy u · v = α · (w · x). Then either [u] = [ w] and [v] = [ x], or 

[u] = [ x] and [v] = [ w].

Here GF (p)× is viewed as the embedded scalar subgroup of GF (p3)×.Equivalently, equality [u][ v] = [ w][ x] in the quotient group is the assertion that u · v = α · (w · x) for some α ∈ GF (p)×; the displayed theorem is this scalar form. The proof splits on whether β = u · w−1 lies in GF( p):

• Base case (β ∈ GF (p)): then [u] = [ w] directly from the coset characterization [a] = [ b] ⇐⇒ a−1b ∈ GF (p)×. The multiplicative relation then forces [v] = [ x].

• Non-base case (β /∈ GF (p)): the intersection V ∩ β−1V is 1-dimensional (Theorems 3.2 and 3.3). Now w ∈ V (given) and βw = u ∈ V , so w ∈ β−1V ; similarly v ∈ V and from 

u · v = α · w · x we compute, using w̸ = 0 and commutativity, 

βv = ( u/w ) · v = ( u · v)/w = α · x ∈ V (since α ∈ GF (p)× and x ∈ V ,the GF (p)-subspace V is closed under scalar multiplication). Thus 

v ∈ β−1V . Hence both w and v lie in V ∩ β−1V , a 1-dimensional space, giving v = c · w for some c ∈ GF( p)×, hence [v] = [ w] and [u] = [ x].

3.3 Combinatorial bridge 

The final step connects the quotient-group Sidon property to the combinato-rial IsSidonMod predicate. This bridge has four components: 1. Projective representatives : we construct p + 1 nonzero elements of 

ker ( Tr ) representing the projective lines of the 2-dimensional kernel, parametrized by Option ( GF (p)) (the p affine lines plus one line at infinity). 82. Injectivity : these representatives map to distinct cosets in the quotient, proved by showing that proportionality of representatives forces equality of the parametrizing indices. 3. Cyclic isomorphism : we exhibit a group isomorphism Z/(p2 + p +1) Z ∼= GF (p3)×/GF (p)× by computing the index [ GF (p3)× : GF (p)×] = (p3 − 1) /(p − 1) = p2 + p + 1 and invoking cyclicity of the quotient (since GF (p3)× is cyclic as the multiplicative group of a finite field, its quotient by any subgroup is cyclic). 4. Additive transfer : let g be a generator of the cyclic quotient of order 

M = p2 + p + 1 , and let i : GF (p3)×/GF (p)× → Z/(p2 + p + 1) Z be the group isomorphism sending [x] 7 → log g[x]. No canonical generator is used; any chosen cyclic generator yields such an isomorphism. Since 

i is a homomorphism, a multiplicative collision [u][ v] = [ w][ x] gives 

i([ u]) + i([ v]) ≡ i([ w]) + i([ x]) ( mod M ), so the multiplicative Sidon property yields modular-additive Sidon: 

M | (i([ u])+ i([ v])) − (i([ w])+ i([ x])) =⇒ { [u], [v]} = {[w], [x]}

=⇒ { i([ u]) , i ([ v]) } = {i([ w]) , i ([ x]) }.

This is formalized via ZMod.intCast_zmod_eq_zero_iff_dvd .Concretely, after choosing the isomorphism i, the integer set S is formed by taking the canonical integer representative in {0, . . . , M − 1} of each residue i([ u]) , where u ranges over the chosen projective representatives of the trace kernel. Injectivity of the projective representatives and of the cyclic isomorphism gives |S| = p + 1 . The passage from equality in 

Z/M Z to the divisibility predicate in IsSidonMod is exactly the equivalence 

x = y ⇐⇒ M | x − y; unordered-pair equality is then transported back through the injective representative map. Thus the final output is genuinely a finite subset of Z, not merely a subset of the quotient group.

# 4 Sidon Set Infrastructure 

4.1 Predicates 

We define three levels of Sidon property, each building on the previous: 

• IsSidon (A): the set A ⊆ Z is Sidon. 

• IsSidonMod (M, A ): A is Sidon modulo M , i.e., M | (a + b) − (c + d)

with a, b, c, d ∈ A implies {a, b } = {c, d }.

• IsIntervalSidon (N, A ): A is Sidon and A ⊆ { 1, . . . , N }.The key structural insight is that the modular predicate serves as the natural intermediate between algebraic constructions (which produce Sidon sets in cyclic groups) and combinatorial targets (which require Sidon sets in integer intervals). 

4.2 Structural cardinality identities 

For finite integer Sidon sets, the library also proves exact structural consequences that are independent of the Singer construction. The theorem 

IsSidon.card_add gives the sharp sumset formula |A + A| = |A|(|A| + 1) /2,and isSidon_iff_card_add proves the converse characterization: maxi-mal sumset cardinality forces the Sidon property. For nonempty Sidon sets, IsSidon.card_sub gives |A − A| = |A|2 − | A| + 1 . Separately, 

IsSidon.addEnergy_eq gives additive energy 2|A|2 −| A| without a nonempti-ness hypothesis. Combining the exact sumset and difference-set formulas, 

IsSidon.card_sub_ge_card_add proves |A − A| ≥ | A + A| for nonempty Sidon sets (strictly for |A| ≥ 3 via IsSidon.card_sub_gt_card_add ).

4.5 Transfer machinery 

The modular-to-interval transfer proceeds via cyclic window relabeling. Given a modular Sidon set S with IsSidonMod M S , a cyclic window of length N

starting at offset u restricts S to elements x with (x − u) mod M < N and relabels them to {1, . . . , N }. The key theorem ( IsSidonMod.windowSidon ): 

Lemma 4.3 (Cyclic window transfer) . If IsSidonMod M S and 0 < N ≤ M ,then for any offset u, the relabeled window restriction is an interval Sidon set in {1, . . . , N }.

The cardinality of the window image depends on how many elements of S

fall in the chosen window. The first averaging lemma counts membership rather than exclusion: for each fixed s ∈ S, exactly N of the M offsets satisfy (s − u) mod M < N . Hence the sum, over all offsets u, of the window cardinalities is |S|N , so some offset captures at least ⌈| S|N/M ⌉ elements (IsSidonMod.exists_large_intervalSidon ). The full-size transfer uses the stronger cyclic-gap structure forced by the modular Sidon property. Sort the residues of S in {0, . . . , M − 1} and let the cyclic gaps be the successive forward differences. These gaps are positive and sum to M . They are also pairwise distinct: if two distinct cyclic gaps were equal, the two associated ordered difference representations would contradict the modular Sidon condition. If G is the largest gap and s = |S|, then the s

distinct positive gaps are all at most G, so 

M ≤ G + ( G − 1) + · · · + ( G − s + 1) = sG − s(s − 1) 2 .

5.1 Upper bound 

Theorem 5.1 (Pair-difference upper bound) . For N ≥ 1, h(N ) ≤ √2N + 1 .

This follows from the pair-difference counting bound: if A is a Sidon subset of {1, . . . , N } with |A| = m, then the m(m−1) /2 positive pairwise differences are all distinct elements of {1, . . . , N − 1}. Thus m(m − 1) /2 ≤ N − 1. Since 

(m − 1) 2 ≤ m(m − 1) ≤ 2( N − 1) < 2N, 

we obtain m − 1 < √2N , hence m ≤ √2N + 1 .We also formalize Lindström’s cross-difference inequality as reusable infrastructure:

(m − k) elements, establishing the injectivity of the cross-difference map via the Sidon property, and bounding the image within {1, . . . , N − 1}.

15 7 Mathematical Lessons from Formalization 

7.1 The projective-line representation was forced by formal-ization 

In the informal Singer argument, one says “the q + 1 lines through the origin in the 2-dimensional trace kernel give q + 1 cosets.” Formalizing this required constructing explicit representatives : a function Option(GF( p)) →

ker ( Tr ) \ { 0} mapping each projective coordinate to a specific nonzero kernel element. The choice of basis is arbitrary but must be fixed : the entire injectivity argument (showing distinct indices map to distinct cosets) depends on the explicit form of the representatives in terms of a chosen kernel basis. This is invisible in informal proofs but becomes a nontrivial formal proof component. 

7.2 The modular Sidon predicate is the natural bridge object 

A recurring design tension in additive combinatorics formalization is whether to work with sets in Z or in Z/M Z. Our experience strongly favors 

IsSidonMod M A (with A ⊆ Z) over direct Z/M Z sets, because: 

• The Singer quotient-group argument naturally produces an Z/(p2 + p +1) Z-indexed result, expressible via divisibility. 

• The interval Sidon target uses subsets of Z.

• The modular predicate bridges both without changing the carrier type. The direction IsSidonMod =⇒ IsSidon is trivially one-line (every literal collision a + b = c + d also satisfies M | 0). The non-trivial direction is the converse: the no-wraparound lemma shows that an interval Sidon set in {1, . . . , N } with M ≥ 2N − 1 is also Sidon modulo M , since pairwise-sum differences lie in {− (2 N − 2) , . . . , 2N − 2} ⊂ (−M, M ), making M -divisibility equivalent to zero. This factoring into “construction → modular → interval” is cleaner than any single-step argument we attempted. 

7.3 The non-invariance argument requires the full irreducibil-ity theory 

Theorem 3.2 (no proper invariant subspace) might appear to be a simple linear algebra fact, but its formalization reveals a substantive mathematical dependency: one needs that the minimal polynomial of α / ∈ GF (p) has degree 3 (irreducibility over the base field), which in turn requires the intermediate field characterization of algebraic elements in Mathlib. The alternative proof strategy—directly showing V̸ = αV by trace computations—turned out to be longer in Lean than the general invariant-subspace argument, because trace manipulation requires unfolding the Galois field representation. 16 7.4 The extremal function is defined non-constructively 

The definition h(N ) = max {| A| : A ⊆ { 1, . . . , N } is Sidon } requires an existence argument for the maximum. We define sidonMaximum via 

Classical.choose , making it noncomputable in Lean. This forces a clean separation: properties of h(N ) are proved via bounds and transfer theorems, never by evaluation. This mirrors the mathematical practice: known asymptotic results about h(N ) proceed through existence arguments (Singer, Bose–Chowla [ 6 ]) and counting bounds rather than explicit construction of optimal sets for general N .

7.5 The conditional structure separates concerns cleanly 

By factoring Erdős 30 into SidonUpperBound + SidonLowerBound , and further decomposing the lower bound into SingerFamily + Subpolynomial-PrimeGap , the formal development achieves a clear separation: 

• Singer family: pure algebra (now discharged). 

• Prime gap transfer: analytic number theory (open hypothesis). 

• Upper bound: formalized unconditionally for ε ≥ 1/4; still a hypothesis for the full range ε > 0.

• Assembly: just the triangle inequality. The formal development makes this decomposition explicit with machine-checked interfaces between the components. 

7.6 Floor-rounding is mathematical, not bookkeeping 

The Singer-via-Bertrand lower bound (Theorem 5.5) is stated in the integer-rounded form h(N ) > ⌊(⌊√N ⌋ + 1) /2⌋ rather than the informal h(N ) ≥√N / 2. This is not bookkeeping: the difference between ⌊√N ⌋ and √N

interacts with the floor-form transfer threshold p2 + p + 2 − ⌊ 3p/ 2⌋ and with the integrality of Sidon-set cardinalities, and the integer-rounded statement is what one actually proves. Likewise, the shift-incidence upper bound (Theorem 5.3) is stated as ⌊√N ⌋ + ⌊

Adjacent Lean resources include finite Sidon/perfect-difference-set coun-terexample formalizations associated with Alexeev–Mixon [ 1 ] and Ho’s for-malized use of Singer difference sets in a high-dimensional distance con-struction [ 13 ] (repository file DiameterConstruction/Singer.lean , commit

8.2 Sidon sets and computation 

Alexeev and Mixon [ 1 ] disproved Erdős’s 1976 conjecture that every finite Sidon set extends to a finite perfect difference set, exhibiting {1, 2, 4, 8, 13 }

as a counterexample and rediscovering an earlier counterexample of Hall. They used a large language model to generate a Lean 4 proof of both counterexamples, which is a small, finite verification. Their contribution is a verified counterexample to a specific extension conjecture; ours is structural algebraic infrastructure for the asymptotic theory. The two approaches are complementary: small-case verification answers specific finite questions, while algebraic formalization builds reusable theory for asymptotic reasoning. Ho [ 13 ] gives a Lean 4 formalization of a counterexample to Erdős’s diameter conjecture for separated distances. That development includes a formalized Singer difference-set input for prime powers as a black-box ingredient of a Euclidean distance construction; the diameter construction does not depend on either an interval transfer or any h(N ) infrastructure. 18 The targets are therefore complementary: Ho’s broader q = pk Singer scope is needed for the diameter-conjecture counterexample, whereas the present formalization develops, around the prime-field Singer construction, the algebraic-combinatorial interface (trace-kernel proof chain, projective-line representatives, modular-to-interval Sidon transfer with the explicit threshold 

p2 + p + 2 − ⌊ 3p/ 2⌋, and the extremal-function infrastructure for h(N )) that Erdős Problem 30 specifically requires. No earlier Lean 4 treatment we have located develops the combinatorial bridge from a Singer-type quotient-group Sidon family to interval Sidon sets through the modular Sidon predicate. 

8.3 AFM context 

The first volume of the Annals of Formalized Mathematics [ 5 , 16 , 17 ] in-cludes formalizations of Fermat’s Last Theorem for regular primes, zeta and L-functions, and adele ring compactness. Our contribution occupies a complementary niche: providing foundational infrastructure in additive combinatorics where the engineering challenge lies in bridging algebraic and combinatorial worlds rather than formalizing a single deep proof. 

# 9 Conclusion 

We have presented a Lean 4 formalization of Singer’s Sidon set construction for prime fields, together with a Sidon set library and conditional Erdős 30 reduction. The development demonstrates that: 1. Classical algebraic constructions require explicit handling of projective representatives, modular-to-interval bridges, and type transitions that informal proofs routinely elide. 2. Formalization forces honest scope accounting: the prime-only limitation of our Singer argument, the distinction between upper-bound and lower-bound halves of Erdős 30, and the exact logical structure of the conditional reduction are all made explicit. 3. The modular Sidon predicate is a natural formalization-friendly inter-mediate between algebraic and combinatorial viewpoints. Thus the submission contributes a checked prime-field Singer construction, modular-to-interval Sidon transfer, formal h(N ) bounds, and a reusable theorem-indexed Lean artifact for future additive-combinatorics formaliza-tion. 

9.1 Future work 

1. Prime power extension : generalize to q = pk by formalizing the field tower GF (pk) ↪→ GF (p3k). A key component would be an intermediate subfield instance connecting GaloisField p (3*k) to GaloisField p k as a degree-3 extension. 19 2. Baker–Harman–Pintz : formalize prime gap results to uncondition-ally discharge the lower bound for ε ≥ 0.2625 ; this requires substantial analytic number theory infrastructure (sieve methods, zero-density estimates, exponential sum techniques) not yet in Mathlib. 3. Fourier and higher-order representation methods : extend the finite representation-function theory into a cyclic Fourier API (Parseval, convolution, and L4 energy) and use it to prove standalone Sidon estimates rather than merely additional infrastructure.
